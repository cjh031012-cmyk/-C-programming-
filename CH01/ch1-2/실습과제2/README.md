## 소스코드 설명

```
#include <iostream>
```

- cin, cout 등 라이브러리 객체의 선언을 포함하고 있는 헤더파일 iostream을 포함한다.

```
int main()
```

- 메인함수 시작

```
std::cout << "Hello";
```

- 화면에 Hello 를 출력한다.  
- std::cout은 화면에 출력해주는 객체이고 << 는 삽입연산자이다.


```
return 0;
```

- 0을 반환하고 메인함수 정상 종료
